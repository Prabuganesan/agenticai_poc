# Node Icons

**GET** `/node-icons`

API endpoint for Node Icons.
 

## Responses

| Status | Description |
| :--- | :--- |
| **200** | Successful operation |
| **400** | Invalid input |
| **401** | Unauthorized |
| **500** | Internal server error |

## Examples

### Response (200)

```json
{
  "success": true,
  "data": {}
}
```
