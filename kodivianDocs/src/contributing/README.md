## üìñ Contribute to Docs

1. Fork the official [Kodivian Docs Repo](https://github.com/KodivianAI/KodivianDocs)
2. Clone your forked repository
3. Create a new branch
4. Switch to the branch you just created
5.  Go into repository folder

    ```bash
    cd KodivianDocs
    ```
6. Make changes
7. Commit changes and submit Pull Request from forked branch pointing to [KodivianDocs main](https://github.com/KodivianAI/KodivianDocs)
