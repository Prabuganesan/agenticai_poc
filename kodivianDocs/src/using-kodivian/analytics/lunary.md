# Lunary

[Lunary](https://lunary.ai/) is a monitoring and analytics platform for LLM chatbots.

Kodivian has partnered with Lunary to provide a complete integration supporting user tracing, feedback tracking, conversation replays and detailed LLM analytics.

Kodivian users can get a 30% discount on the Teams Plan using code `KodivianFRIENDS` during checkout.

Read more on how to setup Lunary with Kodivian [here](https://lunary.ai/docs/integrations/Kodivian).
