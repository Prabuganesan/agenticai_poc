# Autonomous API Docs


A list of Autonomous public APIs, allowing users to programmatically execute many of the same tasks as the same in the GUI.

To run this standalone:

1. Spin up Autonomous server:
    ```sh
    cd autonomous
    pnpm start
    ```
2. Start API Docs server:
    ```sh
    cd packages/api-documentation
    pnpm start
    ```

To run in dev mode:

```sh
cd autonomous
pnpm dev
```

Docs will be served on: `http://localhost:6655/api-docs`
